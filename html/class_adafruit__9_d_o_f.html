Populates the .roll/.pitch/.heading fields in the sensors_vec_t struct with the right angular data (in degree).

The starting position is set by placing the object flat and pointing northwards (Z-axis pointing upward and X-axis pointing northwards).

The orientation of the object can be modeled as resulting from 3 consecutive rotations in turn: heading (Z-axis), pitch (Y-axis), and roll (X-axis) applied to the starting position.

Parameters
accel_eventThe sensors_event_t variable containing the data from the accelerometer
mag_eventThe sensors_event_t variable containing the data from the magnetometer
orientationThe sensors_vec_t object that will have it's .roll, .pitch and .heading fields populated
Returns
false if unable to read event data from the sensor
